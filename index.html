Getting Started

Start by downloading the repository or by cloning the project: 

git clone https://github.com/jacobxperez/essentials.git

Default Build

The easiest way to include Essentials in your projects is by copying the pre-compiled CSS file into your project folder and then link to it in HTML.

<link rel="stylesheet" href="css/style.css">
  • /css
    • style.css

Custom Build

Adjust the variables and if needed, add custom modules then compile everything together in style.less to produce the final CSS. Optional, comment out unwanted components from style.less then compile your project. Check out Editors and Plugins for compiling LESS.

  • /css/less
    • modules (add custom components)
    • custom.less (write or import custom modules)
    • style.less (compile everything together)
    • variables.less (adjust the settings)
