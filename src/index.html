About

RAMs is a CSS framework with Zero-Class Architecture that utilizes the power of data-attributes for styling components. By removing class names from CSS, your HTML becomes less cluttered easy to read, and more scalable.

Features

  • Zero-Class Architecture
  • Powered by data-attributes
  • Dark mode enabled
  • Built with less, but better

Getting Started

Start by downloading the repository or by cloning the project: 

git clone https://github.com/jacobxperez/rams.git

Default Build

Copy the pre-compiled CSS file into your project folder and then linking to it in HTML. 

<link rel="stylesheet" href="css/main.less">
  • /css
    • main.css

Custom Build

Edit the variables, add custom modules, then compile main.less to produce the final CSS. Check out Editors and Plugins for compiling LESS.

  • /less
    • core (important files for compiling)
    • modules (add custom modules here)
    • main.less (compile to produce the final CSS)
    • var.less (variables for font, colors, and more)
